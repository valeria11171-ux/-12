## Описание задачи

Модель определяет, будет ли заказ доставлен вовремя, на основе следующих параметров:
- ID ресторана
- Количество товаров в заказе
- Стоимость заказа
- Расстояние доставки
- Планируемое время приготовления
- Час оформления заказа
- Выходной день

## Архитектура решения

### Модель
- **Алгоритм:** Random Forest Classifier
- **Библиотека:** scikit-learn
- **Метрики:** Accuracy ~0.87, F1-score ~0.85

### Сервис
- **Фреймворк:** FastAPI
- **Сервер:** Uvicorn
- **Формат данных:** JSON

### Предобработка данных
- Кодирование категориальных признаков (LabelEncoder)
- Масштабирование числовых признаков (StandardScaler)
